ph-jaxb22-plugin
================

JAXB 2.2 plugin that adds some commonly needed functionality

#Maven usage
Add the following to your pom.xml to use this artifact:
```
<dependency>
  <groupId>com.helger</groupId>
  <artifactId>ph-jaxb22-plugin</artifactId>
  <version>2.2.7</version>
</dependency>
```
